Problem 1:
Create a function called div23(n) which will take a single non-negative integer and return True if it's evenly divisible by both 2 and 3, and False otherwise. 
            
    def div23(n):
      if n % 2 == 0 and n % 3 == 0:
        return True
      return False
            
          
             
    div23(4492738) -> False
Problem 2:
Create the function div23_between(low,high) that will be given 2 non- negative integers, with low <= high and will return the number of integers between low and high (including low and high) that are divisible by both 2 and 3. 
            
    def div23_between(low, high):
      c = low
      num = 0
      while c <= high:
        if div23(c):
          num += 1
        c += 1
      return num 
            
          
            
    div23_between(0, 399154) -> 66526
Problem 3:
Create the function sum_dig(n) that adds up the digits of n, where n is a non-negative number less than 1,000. 
            
    def sum_dig(n):
      c = n
      sum = 0
      while c > 0:
          sum += c % 10
          c /= 10
      return sum
            
          
            
    sum_dig(667) -> 19
Problem 4:
How many integers between 0 and 999 have the following property: that the sum of its digits is evenly divisible by 11? 
            
    def how_many_11():
      c = 0
      ans = 0
      while c <= 999:
          if sum_dig(c) % 11 == 0:
              ans += 1
          c += 1
      return ans
            
          
            
    how_many_11() -> 91
