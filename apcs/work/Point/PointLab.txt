Create the data type Point that represents a pair of integers (x,y)
situated in a two dimensional coordinate system.

1.  Declare the instance variables _x and _y.

2.  Create the default constructor.

3.  Write the accessor methods:
     int getX()
     int getY()

4.  Overwrite the toString method.



5.  Write the modifier method void move(int x, int y).


6.  Overload the constructor by using the heading Point(int x, int y).

7.  Overwrite the equals method.

8.  Write the accessor methods:
	Point getLocation()

9.  Overload the constructor again. Use the heading Point(Point other).

10.  Use StdDraw, implement the method void draw().

11. Write the modifier method Point translate(int dx, int dy).


11. Write the modifier method void setLocation(Point other).
