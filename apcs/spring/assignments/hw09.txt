4. Write the body of the reverse function, that uses a Stack to reverse the 
   elements in a List.

   public static void reverse(List<Integer> L){
	StackArray S = new StackArray(L.size());

	while (L.size() > 0)
		S.push(L.remove(0));

	while (S.size() > 0)
		L.add(S.pop());

   }
